A Python library for interacting with the LXD REST API.

Installation
=============
.. code-block:: console

  # Create a virtual environment
  python3 -m venv myvenv
  cd myvenv

  # Activate the virtual environment
  . bin/activate
  # Install pyLXD in it
  pip install pylxd
